## 출처: 연구개발계획서 1~3장 / (2) 국내외 메인베어링 기술 현황 및 필요 기술 도출

## 1. 설계 기술 (2-1)

### 1-1. 국제 설계 규격의 구조적 한계

- 현재 풍력 베어링 설계의 기반: **ISO 281** 및 **ISO/TS 16281**
  - IEC 61400 등 각종 풍력터빈 규격에서도 ISO 기반 메인베어링 설계 요구
  - 기본 하중 계산 및 수명 산정에는 합리적 프레임 제공

- **한계 1 — 표면 기점 피로 고려 한계**
  - ISO 규격은 **내부 기점 피로(Subsurface-Initiated Fatigue)** 기반 수명 산정
  - 재료 내부 응력 분포·피로 한도를 주요 변수로 사용 → 윤활 양호 시 합리적
  - 실제 대형 풍력 메인베어링 파손은 **표면 기점 피로(마이크로피팅 등)** 가 지배적
  - 표면 기점 손상은 윤활막 두께·표면 거칠기·미끄럼 조건·오염 등 복합 현상에 의존
  - → ISO 규격만으로는 포착 불가능

- **한계 2 — 제조 변동성 고려 한계**
  - 대형 베어링의 소재 편차, 열처리 경도 분포 불균일, 가공 공차·표면 상태 변화
  - → 실제 하중 분포·응력 상태에 직접 영향
  - → 국제 규격은 이러한 제조 변동성을 구체적으로 반영하지 않음
  - → 설계 수명 vs. 실제 수명 간 괴리 발생

### 1-2. 국외 선진업체 설계 기술 현황 (SKF, Schaeffler, NSK 등)

> ISO 기반 설계 + ISO 규격의 구조적 한계를 체계적으로 보완

#### 표면 기점 피로 해석 기술

**SKF — GBLM (Generalized Bearing Life Modeling)**
- 실제 표면 형상 및 윤활 조건에 따른 표면 열화 현상을 정교한 수치해석 모델로 계산
- 트라이볼로지 기반 수명 해석 모델

**Schaeffler — BEARINX (자체 개발 베어링 해석 소프트웨어)**
- 윤활 조건, 오염도, 실제 접촉 압력이 표면 기점 피로에 미치는 영향 반영
- 수명 해석 수행

|------|------|
| KMT Lidköping **VTG4000** | 대형 연삭·복합가공기 |
| Danobat **DVG/VG** | 대형 연삭·복합가공기 |
| Supfina **RacePro 4000** | 대형 슈퍼피니싱 장비 |

- **대형 링 단일 셋업(One-setup) 복합 가공** — 기하학적 정밀도 확보

### 1-3. 국내 기술 현황 (설계)

- 요(yaw)·피치(pitch) 베어링 설계 경험 있음, **메인베어링 설계 경험 전무**
- **해외 기술에 전적으로 의존**
- 설계 측면: ISO 규격 기반 기본설계 수준에 머무름
  - 표면 기점 손상 메커니즘 반영 수명 해석 역량 미확보
  - 트라이볼로지 기반 상세설계 역량 미확보
  - 피로해석용 하중 데이터 도출 기술 미확보
- 제조 측면: 대형 메인베어링 제조기술 수준 자체가 체계적으로 미파악
  - 국외 선진업체 수준의 대형 연삭·복합가공, 슈퍼피니싱 전용 장비 구축 미비

## 3. 슬라이드 활용 방향 메모

### 설계 기술 (국외) 핵심 포인트
- ISO 한계 인식 → SKF/Schaeffler가 자체 해석 도구(GBLM, BEARINX)로 보완
- 설계뿐 아니라 **전용 제조 장비까지 갖춰야 설계-제조 정합성** 확보 가능
- 국내는 이 두 가지(해석 도구 + 제조 장비) 모두 부재

### 시험 인프라 (국외) 핵심 포인트
- SKF 4,000만 유로 / Schaeffler 700만 유로 — **대규모 투자**가 필요한 영역
- 국내: 2026년 이후 인프라 구축 예정이나 **시험 절차 기술 자체가 없음**
- → 본 과제에서 시험 절차 및 신뢰성 평가 체계 개발 필요성 근거
