# 기술개발 필요성 내용 정리
## 출처: 연구개발계획서 / 기술개발 필요성 (설계·시험 역량 부재 → 기술종속 → 개발 필요)

---

## 핵심 논리 구조

```
국내 메인베어링 설계·시험 역량 부재
        ↓
해외 선진사에 전적 의존 + 핵심 설계 데이터 블랙박스화
        ↓
정보 비대칭 · 기술 종속 · 비용 전가 구조 고착
        ↓
독자 설계 검증 역량 + 시험·인증 기술 확보 필요
```

---

## 1. 설계 기술 개발 필요성

### 1-1. 현재 상황: 기술 종속과 정보 비대칭

**① 해외 선진사 전적 의존**
- 국내 메인베어링 설계 경험 전무 → 조달·적용 모두 해외 제품 의존
- 요(Yaw)·피치(Pitch) 베어링 설계 경험은 있으나, 메인베어링은 이와 구조적으로 상이한 별도 기술 영역
- 국내 기업이 보유한 설계 역량은 ISO 규격 기반 기본설계 수준에 한정

**② 핵심 설계 데이터의 블랙박스화**
- 해외 제조사는 메인베어링의 핵심 설계 파라미터(접촉 기하, 내부 하중 분포, 표면처리 조건, 윤활 설계 등)를 비공개로 관리
- 국내 수요 기업은 설계 근거 없이 납품받는 구조 → 베어링 선정·검증 능력 부재
- 파손 발생 시 원인 분석조차 독자적으로 수행 불가

**③ 미확보 핵심 설계 역량**

| 설계 역량 | 국외 선진사 | 국내 현황 |
|----------|------------|---------|
| 피로해석용 하중 데이터 도출 | 확보 | **부재** |
| 표면 기점 피로 수명 해석 (트라이볼로지 기반) | 확보 (SKF GBLM, Schaeffler BEARINX 등) | **부재** |
| 제조 변동성 고려 상세설계 | 확보 | **부재** |
| 그리스 윤활 해석 기술 (SPH 등) | 확보 | **부재** |

---

### 1-2. 기술 종속이 초래하는 구체적 문제

**① 해외 제조사의 책임 회피 구조**
- 파손 발생 시 해외 제조사는 "운용 조건 이탈", "설치 불량" 등을 이유로 제품 결함 책임 회피 가능
- 국내 측은 설계 데이터 미보유로 반박 근거 제시 불가 → 비용 전가 수용 불가피

**② 교섭력 부재**
- 기술 검증 능력 없이는 가격·품질·납기에 대한 대등한 협상 불가
- 공급 독점 구조에서 조달 리스크(납기 지연, 단종, 가격 급등) 상시 노출

**③ 장기적 기술 발전 차단**
- 독자 설계 역량 없이는 20MW+급 대형화에 따른 베어링 설계 요구 사항 대응 불가
- 국내 풍력산업 공급망 고도화의 핵심 병목

---

### 1-3. 필요한 설계 역량 개발 방향

**① 독자 설계 검증 역량 확보**
- 피로해석용 하중 데이터 도출: 실제 운전 하중 스펙트럼 기반 등가 하중 산출 기술
- 표면 기점 수명 해석: 트라이볼로지(Tribology) 기반 접촉 피로 해석 (ISO 한계 보완)
- 제조 변동성 반영 상세설계: 열처리 경도 분포·가공 공차·표면 조도 변동을 설계에 통합

**② 기술적 교섭력 및 엔지니어링 데이터 확보 목표**
- 설계 검증 역량 → 해외 제조사와 대등한 기술 대화 가능
- 자체 검증 데이터 축적 → 파손 원인 분석·책임 규명 근거 확보
- 장기적으로 국산화 가능한 설계 기술 기반 마련

---

## 2. 시험 기술 개발 필요성

### 2-1. 현재 상황: 인프라 구축 중, 방법론 전무

**① 국내 시험 인프라 단계적 구축**

| 기관 | 내용 | 시기 |
|------|------|------|
| 경남테크노파크 | 15MW급 풍력 나셀 테스트베드 구축 | 2026년 완공 예정 |
| 경북테크노파크 | 초대형 해상 풍력터빈 테스트베드 설계 연구 | 진행 중 |

- 인프라 구축은 진행되고 있으나, **메인베어링에 특화된 시험 절차 및 인증 방법론은 부재**
- 20MW+급 대형 메인베어링에 대응 가능한 전용 시험 인프라 미확보

**② 해외 시험 인프라와의 격차**

| 기관 | 투자 규모 | 시험 가능 규모 |
|------|----------|--------------|
| SKF — Sven Wingquist Test Centre (독일) | 약 4,000만 유로 | 외경 6m 이하 대형 베어링 |
| Schaeffler + LORC — Astraios (덴마크) | 약 700만 유로 | 25MW+급 메인베어링 |
| Clemson University (미국) | — | 15MW급 드라이브트레인 |
| ORE Catapult (영국) | — | 15MW→28MW급 나셀 |

→ 해외는 수천만 유로 투자로 실스케일(Full-scale) 시험 인프라를 구축·운영 중
→ 국내는 구축 예정 인프라는 있으나 **이를 활용한 시험 기술 자체가 없음**

---

### 2-2. 핵심 시험 기술 부재의 문제

**① 설계 검증 수단 부재**
- 메인베어링 설계 결과물의 신뢰성을 실험적으로 검증할 수단 없음
- 설계 → 제작 → 현장 투입 전 요소 수준 시험·성능 확인 불가

**② 인증 체계 부재**
- 국내 풍력 메인베어링에 대한 시험 인증 절차 미수립
- 해외 인증(GL, DNV 등)에 전적 의존 → 인증 비용·기간·조건 모두 해외 주도

**③ 가속수명시험 설계 기술 부재**
- 실제 20년 운용 조건을 단기간으로 압축하는 가속수명시험(ALT) 설계 역량 미확보
- 시험 하중 스펙트럼 설정 방법(등가 손상 환산) 기술 부재

---

### 2-3. 필요한 시험 기술 개발 방향

**① 시험 하중 스펙트럼 설정 기술**
- 실제 해상 풍력터빈 운전 데이터 기반 하중 스펙트럼 정의
- 피로 등가 하중 산출 → 시험 조건 설정 방법론 확립

**② 가속수명시험(ALT) 설계 기술**
- 실운전 20년 → 시험 기간 내 등가 손상 재현을 위한 가속 계수 산출 방법
- 마이크로피팅·표면 기점 피로 등 주요 파손 모드별 가속 시험 설계

**③ 시험 인증 절차 개발**
- 국내 메인베어링 시험 인증 표준 절차 수립
- 향후 구축되는 국내 테스트베드(경남TP 등)와 연계 가능한 인증 체계 설계

## 3. 종합: 기술개발 필요성 핵심 논거

### 설계 측면 요약

> 국내는 메인베어링 설계 경험이 전무하고 핵심 설계 데이터에 접근할 수 없어, 해외 제조사의 기술 종속·비용 전가 구조에 무방비 상태임. 독자 설계 검증 역량 확보 없이는 대등한 기술 교섭도, 20MW+급 대형화 대응도 불가능함.

### 시험 측면 요약

> 국내 시험 인프라는 단계적으로 구축되고 있으나, 이를 활용할 시험 기술(하중 스펙트럼 설정, 가속수명시험, 인증 절차)이 전무한 상태임. 하드웨어(인프라) 구축과 소프트웨어(방법론) 개발을 병행하지 않으면 인프라가 있어도 검증·인증 수행 불가.

### 양자의 연계성

```
설계 기술 개발 ──→ 설계 파라미터 도출 ──→ 시험 조건 정의 (하중 스펙트럼)
                                                    ↓
                              가속수명시험 설계 + 시험 인증 절차
                                                    ↓
                              신뢰성 검증 데이터 → 설계 피드백 (반복 개선)
```

→ 설계 기술과 시험 기술은 독립적이 아닌 **상호 의존적 역량**: 어느 한쪽만 확보해서는 완전한 국산화 불가

---

## 4. 슬라이드 활용 방향 메모

| 논거 | 핵심 키워드 | 시각화 방향 |
|------|-----------|-----------|
| 기술 종속 구조 | 블랙박스, 정보 비대칭, 교섭력 부재 | 국내↔해외 비대칭 다이어그램 |
| 설계 역량 부재 매트릭스 | 5개 기술 모두 부재 | 표 또는 체크리스트 (✗ 강조) |
| 시험 인프라 갭 | 인프라 구축 중 vs 방법론 전무 | 타임라인 + 갭 표시 |
| 설계·시험 연계 필요성 | 상호 의존 구조 | 순환 피드백 다이어그램 |
